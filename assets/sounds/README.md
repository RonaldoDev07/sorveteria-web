# Sons do Aplicativo

## Arquivos Necessários

Adicione os seguintes arquivos de som nesta pasta:

### 1. success.mp3
- **Quando tocar**: Venda concluída, cadastro salvo, pagamento recebido
- **Sugestão**: Som de "ding" ou "cash register"
- **Duração**: 0.5-1 segundo
- **Sites para baixar sons grátis**:
  - https://freesound.org/
  - https://mixkit.co/free-sound-effects/
  - https://www.zapsplat.com/

### 2. error.mp3
- **Quando tocar**: Erro na operação, validação falhou
- **Sugestão**: Som de "buzz" ou "error beep"
- **Duração**: 0.3-0.5 segundo

### 3. click.mp3
- **Quando tocar**: Click em botões (opcional)
- **Sugestão**: Som de "click" suave
- **Duração**: 0.1-0.2 segundo

## Como Adicionar

1. Baixe os sons nos sites acima
2. Renomeie para: `success.mp3`, `error.mp3`, `click.mp3`
3. Coloque nesta pasta
4. Rebuild o app

## Alternativa Rápida

Use sons do sistema do celular (já implementado como fallback)
